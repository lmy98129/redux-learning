# react-scaffold
A self-made react-scaffold for react apps with webpack 4.x, babel, hot module replacement, etc.

## Features
* webpack 4.x
* babel
* product/develop environments
* hot module replacement (webpack-dev-server)
* less/sass/css & images files loader
